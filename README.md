# Fishdom

A lightweight proof-of-concept React + TailwindCSS website for an all-in-one fishkeeping service company.

## Features
- Hero section with clear positioning and call-to-action
- Aquarium setup cards with imagery and pricing placeholders
- Services overview cards
- About section
- Contact form
- Dark mode toggle with saved preference

## Tech Stack
- React (Vite)
- TailwindCSS

## Run Locally
```bash
npm install
npm run dev
```

## Build for Production
```bash
npm run build
npm run preview
```
